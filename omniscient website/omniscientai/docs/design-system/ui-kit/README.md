# Omniscient AI — Marketing Website UI Kit

A proposed overhaul of `omniscientai.io`, built from the brand system in the project root.

## Files
- `index.html` — full one-page site wiring all sections together
- `components/Nav.jsx` — sticky translucent top nav with primary CTA
- `components/Hero.jsx` — 55/45 copy/graphic split, lifted from the social hero card
- `components/Pillars.jsx` — the three practice areas (Health · Defense · Agentic ops)
- `components/Services.jsx` — card grid for workshops & engagements
- `components/DarkPitch.jsx` — ink-section pacing block
- `components/CaseStudies.jsx` — three case-study tiles, one featured
- `components/CTAStrip.jsx` — book-a-call strip
- `components/Footer.jsx` — three-column footer with wordmark

## Caveats
Built from brand assets only — no codebase or Figma was provided. The proposed structure is intentionally more ambitious than the current site, which is nearly empty. Copy is drawn from the social hero card, the founder's brief, and the site title tag. Replace with vetted copy before shipping.
